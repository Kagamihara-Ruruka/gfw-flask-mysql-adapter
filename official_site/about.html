Phil Hartmeyer / NOAA · The Ocean Agency
01 / CONTEXT

海域議題讓資料需求變得具體

日菲經濟海域談判與台灣周邊海域劃界，牽動近海漁業、執法權與開發權。 對團隊而言，這讓「海洋資料工程」不只是技術練習，而是海事治理的資料前置作業。

02 / PROBLEM

資料很多，不代表能直接使用

NASA 與 GFW 的來源、格式、解析度、欄位語意都不同。 若沒有來源驗證、清洗、AOI 篩選、網格標準化與查詢服務，前端只能面對難以使用的原始巨量資料。

03 / OBJECTIVE

把異質資料治理成湖倉管線

專題透過 Bronze / Silver / Gold 分層治理，把原始資料轉成可稽核、可追溯、 可被 API 與前端消費的海洋時空資料平台。
