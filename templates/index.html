Frame 補間

Layer crossfade 只修飾兩張真實 snapshot 的圖層替換；關閉時直接切換，不改變資料時間軸。

視覺效果

淡入淡出只修飾圖層替換；高斯模糊只作為縮放 / LOD 重算時的短暫遮罩，不參與播放時間線，也不等待補間。

共同網格策略

    AISStream 金鑰來源 在瀏覽器開啟

    在此 MVP 中，「交付到爬蟲」會寫入本機爬蟲交接檔，供獨立 AIS 收集器使用。請勿把它視為前端直接存取 AIS；UI 只證明自己正在讀取由相同 AIS 爬蟲維護的 SQL table。儲存後不會回傳原始金鑰。
